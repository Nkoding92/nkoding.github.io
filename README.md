# NKoding Website (GitHub Pages)

## How to publish
1) Create a repository named **nkoding.github.io** in your GitHub account.
2) Upload all files from this folder to the repository root.
3) After you commit, your site will be live at **https://nkoding.github.io** within ~1 minute.

## Replace assets
- Put your white NKoding logo file at **/assets/logo-white.png** (exact name).
- Optionally add **/assets/favicon.png** (32×32 or 64×64 PNG) and **/assets/og-image.png** (1200×630).

## Edit content
- Open **index.html** and change the email and any text you want.
